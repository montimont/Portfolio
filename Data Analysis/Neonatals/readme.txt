This dataset focuses on neonatal deaths (nonspecific) in North America. Null values were cleaned up with 0 for a more consistent visualization. The data visualization was done using Tableau, and a link to the Tableau page is provided below.

While originally considering terrorism attacks, I decided to focus on this dataset to better showcase my data visualization skills. The dataset provides insights into neonatal deaths in North America, which can be useful for healthcare professionals and researchers.
